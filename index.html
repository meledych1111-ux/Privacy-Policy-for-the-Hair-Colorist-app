ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ

1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1. Настоящая Политика конфиденциальности («Политика») определяет порядок обработки и защиты персональных данных пользователей мобильного приложения «Hair Colorist» («Приложение»), разработанного Чиграковой Анастасией Сергеевной («Оператор»).

1.2. Оператор обязуется соблюдать конфиденциальность персональных данных Пользователя в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1.3. Используя Приложение, Пользователь выражает своё согласие с условиями настоящей Политики.

2. КАКИЕ ДАННЫЕ МЫ СОБИРАЕМ

2.1. Приложение «Hair Colorist» разработано с минимальным сбором данных. Мы собираем только ту информацию, которая необходима для функционирования Приложения:

  • Технические данные устройства:
    • Тип и версия операционной системы
    • Модель устройства
    • Уникальный идентификатор устройства (IDFA/Android ID/AAID)
    • Версия Приложения
    • Язык и регион устройства
  • Данные, вводимые пользователем для расчётов:
    • Тип волос (натуральные, окрашенные, седые и т.д.)
    • Текущий и желаемый цвет волос
    • Длина волос
    • Процент седины
    • Выбранный бренд краски (ESTEL и др.)
    • Количество продукта (если указано)
  • Данные об использовании (аналитика и реклама):
    • Частота использования различных функций
    • Продолжительность сессий
    • Информация об ошибках и сбоях (логи)
    • Данные для таргетированной рекламы (без персональной идентификации)

2.2. Приложение НЕ собирает и НЕ передаёт: Фотографии и изображения пользователей, геолокационные данные, контактную информацию, данные платёжных карт, данные из адресной книги или социальных сетей (за исключением случаев добровольной авторизации через ВКонтакте).

3. ЦЕЛИ СБОРА ДАННЫХ

3.1. Собранные данные используются исключительно для следующих целей:

  • Обеспечения корректной работы Приложения и выполнения расчётов;
  • Улучшения функциональности и пользовательского опыта;
  • Анализа использования Приложения через сервисы аналитики;
  • Показа релевантной рекламы для монетизации бесплатной версии;
  • Выявления и устранения ошибок, технических проблем;
  • Соблюдения требований действующего законодательства РФ.

4. ОБРАБОТКА И ХРАНЕНИЕ ДАННЫХ

4.1. Все расчёты производятся локально на устройстве Пользователя. Введённые данные для расчётов не передаются на серверы Оператора и не покидают устройство.

4.2. Для сбора аналитики и показа рекламы могут использоваться сторонние сервисы:

  • Яндекс.Метрика — для анализа использования Приложения;
  • ВКонтакте (VK SDK) — для авторизации по желанию пользователя и аналитики;
  • Huawei Mobile Services (HMS Core) — для устройств без сервисов Google, включая Huawei Ads Kit, Huawei Analytics Kit, Huawei In-App Purchases;
  • Данные рекламных сетей — в соответствии с политиками соответствующих платформ.

4.4. Оператор принимает необходимые организационные и технические меры для защиты данных от неправомерного доступа, уничтожения, изменения, блокирования, копирования и распространения.

5.1. Оператор не продаёт, не обменивает и не передаёт персональные данные Пользователей третьим лицам, за исключением:
